# Advanced CLI Calculator in C

## Overview
A robust, command-line based calculator written in C. It performs basic arithmetic operations (addition, subtraction, multiplication, and division) using a modular function approach. 

**Development Note:** This project is a product of 100% original logic building. It was completely self-created in VS Code with an honest use of 0% AI assistance.

## Features
- **100% Original Logic:** Built entirely from scratch without any AI help.
- **Modular Architecture:** Utilizes a dedicated function (`calculator`) for mathematical computations.
- **Input Validation:** Safely checks all user inputs to prevent crashes and filters out invalid characters.
- **Division by Zero Protection:** Actively catches and prevents division by zero at the main execution level.
- **Floating-Point Precision:** Supports decimal calculations for more accurate results.

## Prerequisites
- GCC Compiler (or any standard C compiler)
- Visual Studio Code (IDE used for development)

## How to Compile and Run
1. Open your terminal or command prompt.
2. Compile the code using GCC:
```bash
   gcc calculator.c -o calculator
