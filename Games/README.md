# C Logic Games Collection

This folder contains various logic-based and interactive console games developed in C language. The primary goal of these mini-projects is to strengthen core programming concepts, including control structures, random number generation, loops, and robust user input validation.

## Games Index

| Project Name | Description | Status |
| :--- | :--- | :--- |
| **Number Guessing Game** | A logic game where the user guesses a randomly generated number. | Completed |
| **Snake Water Gun** | A C implementation of the classic interactive game against the computer. | Completed |

**Note:** Each game folder contains its own specific `README.md` with instructions on how to compile and run the program.
