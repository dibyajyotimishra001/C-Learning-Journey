# Leap Year Checker

A simple, efficient C program to determine if a given year is a leap year.

## Description
This program prompts the user to input a specific year and calculates whether it is a leap year based on the standard mathematical rules of the Gregorian calendar:
- A year is considered a leap year if it is perfectly divisible by 4, but not divisible by 100.
- An exception to this rule is that years divisible by 400 are also leap years.

## Prerequisites
- A C compiler installed on your system (such as GCC).
