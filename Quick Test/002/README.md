# ATM Simulator

A simple ATM Simulator built in C that allows users to perform basic banking operations through a menu-driven interface.

## Features

- Check account balance
- Deposit money
- Withdraw money
- Insufficient balance protection
- Input validation using `scanf()`
- Simple and interactive menu system

## Tech Stack

- C Programming Language
- Standard I/O Library (`stdio.h`)

## How to Run

Compile:

```bash
gcc ATM_Simulator.c -o ATM_Simulator
```

Run:

```bash
./ATM_Simulator
```

On Windows:

```bash
ATM_Simulator.exe
```

## Example Menu

```text
1. Check Balance
2. Deposit Money
3. Withdraw Money
4. Exit
```
